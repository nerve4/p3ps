Python3 Project Skeleton for the daily ops.


## Run Nosetests

Make sure pip3, virtualenv vand nose have been installed:
```
pip3 list
pip3 install virtualenv
pip3 install nose
```

Run nosetests:
```
nosetests3
```
output:
```
.
----------------------------------------------------------------------
Ran 1 test in 0.010s

OK
```
